# CakeBakery (React + Tailwind)

A responsive single-page bakery website built with **React.js** and **Tailwind CSS**

## Features
- Single-page layout with multiple sections (Home, About, Products, Blog, Contact) 
- Fully responsive UI (mobile + desktop)
- Reusable React components and clean folder structure 

## Tech Stack
- React.js 
- Tailwind CSS 

## Getting Started
1. Clone the repo
2. Install dependencies: `npm install`
3. Run the project: `npm run dev`
